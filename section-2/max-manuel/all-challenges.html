All challenges for the bootcamp.

The first challenge, as shown on the daily challenge page, will be to build a first website. HTML and CSS basics will be explored.

The second challenge will be to dive deeper into HTML and CSS to explore more concepts so as to lay a solid foundation needed for future web development tasks or projects.

To give opportunity for review of the concepts learnt up to this point, there will be a task that will offer the chance to practice everything learnt about HTML and CSS up to this point.

The first challenge involving serving(hosting) a website will follow. A simple website will be hosted in this section. In an upcoming section a more complex website and web application will be hosted.

Version control with Git and GitHub will be introduced next. A wide range of topics will be covered to provide a good understanding of what employing version control in projects looks like.

Returning to HTML and CSS; layouts and positioning will be covered in-depth.

The important aspect of responsive web design will be covered in this section.
